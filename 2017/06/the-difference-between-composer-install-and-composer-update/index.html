Composer Install和composer Update的区别

2017-06-16

composer install 和 composer update 区别：

install 的时候如果发现 composer.lock 中存在锁定版本，就会安装该版本。不存在的话会安装.json里的版本并创建.lock文件

update 是更新所有依赖的版本并更新composer.lock 文件。
