Php中几个常用的排序算法

时间： 2017-08-02   |   分类： 小知识     |   阅读： 285 字 ~1分钟 
// 冒泡，从第一个开始往后比，再从第二个开始比较到倒数第二个
// 循环
function bubbleSort($numbers) {
    $count= count($numbers);
    if ($count <= 0) return false;
    for ($i = 0; $i < $count; $i++) {
        for ($j = 0; $j < $count- 1 - $i; $j++) {
            if ($numbers[$j] > $numbers[$j + 1]) {
                list($numbers[$j], $numbers[$j +1]) = [$numbers[$j + 1], $numbers[$j]];
            }
        }
    }
 
    return $numbers;
}
 
$num = array(20, 40, 60, 80, 30, 70, 90, 10, 50, 0);
var_dump(bubbleSort($num));

Html中的input File在Chrome中弹出缓慢

时间： 2017-07-10   |   分类： 小知识     |   阅读： 169 字 ~1分钟

<input type="file" accept="image/*"> 改为如下:
<input type="file" accept="image/gif,image/jpeg,image/jpg,image/png">

accept="image/*"属性会对每一个文件都遍历一次所有的"image/*"文件类型，当文件较多时，文件的检验时间较长。

如果是accept="audio/*"和accept="video/*"，同样指明所需文件的类型MIME类型
