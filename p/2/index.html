windows下的php环境安装

时间： 2019-12-01   |   分类： 瞎折腾     |   阅读： 1221 字 ~3分钟

Apache

这个是 apache conf 的内容

# NOTE: Where filenames are specified, you must use forward slashes
# instead of backslashes (e.g., "c:/apache" instead of "c:\apache").

安装服务（需要用管理员权限）:  httpd -k install
卸载服务   httpd -k uninstall
启动服务   httpd -k start
停止服务   httpd -k stop
重启服务   httpd -k restart
查看版本   httpd -V
查看支持的命令 httpd -h

动态添加的html无法使用bootstrap的tooltip

时间： 2018-09-13   |   分类： 笔记     |   阅读： 315 字 ~1分钟

经常有一些数据是通过 ajax 获取的，然后渲染到页面上，通过 append 或者其他方式把 html 加载到 DOM 页面，但这个时候无法使用 js 中的一些事件，可以通过$('body').on(''click'',xxxx,function(){})来解决部分（比如click）事件，但碰到 tooltip 怎么解决

bootstrap 中的 tooltip 调用方式为：$('[data-toggle="tooltip"]').tooltip()

解决方式：

$("body").tooltip({
    selector: '[data-toggle="tooltip"]'
});

本地环境伪造ssl证书

时间： 2018-06-08   |   分类： 瞎折腾     |   阅读： 95 字 ~1分钟
  1. 看到报错：curl: (60) SSL certificate problem: unable to get local issuer certificate
  2. 下载证书 https://curl.haxx.se/ca/cacert.pem ，一般放在D:/wamp64/bin/php/php7.1.9/extras/ssl/cacert.pem
  3. 配置php.ini
curl.cainfo = "D:/wamp64/bin/php/php7.1.9/extras/ssl/cacert.pem"`
openssl.capath="D:/wamp64/bin/php/php7.1.9/extras/ssl/cacert.pem"
  1. 重启服务
